## 🎯 Sobre o Projeto
O **Save Food** é uma solução mobile desenvolvida em Flutter com o objetivo de combater o desperdício de alimentos em ambientes universitários e estabelecimentos locais. O aplicativo conecta estabelecimentos que possuem excedentes de produção a estudantes que buscam alimentação de qualidade a preços reduzidos.

### 🌍 Objetivos de Desenvolvimento Sustentável (ODS)
Este projeto está alinhado com a **Agenda 2030 da ONU**, especificamente:
*   **ODS 2 - Fome Zero e Agricultura Sustentável:** Facilitando o acesso a alimentação acessível.
*   **ODS 12 - Consumo e Produção Responsáveis:** Mitigando o desperdício de alimentos (Meta 12.3).

## ✨ Funcionalidades Principais
- [x] **Seleção de Perfil:** Fluxo de cadastro diferenciado para Estudantes e Estabelecimentos.
- [x] **Autenticação:** Sistema de Login e Registro com validações.
- [x] **Feed de Ofertas:** Listagem dinâmica de produtos em oferta (atualmente via Mock Data).
- [x] **Visualização de Detalhes:** Informações completas sobre o produto, local e horário de retirada.
- [x] **Design Moderno:** Interface responsiva e intuitiva focada na experiência do usuário.
